ESO Update 5 added "Undaunted Enclaves" to the game, guild headquarters for the Undaunted members from which they challenge adventurers to undertake daring missions into the game's many 4-player instanced dungeons. These instanced dungeons feature several unique item sets which players can obtain by defeating the bosses which dwell within and completing Pledges, challenges issued by Undaunted leaders, each day. These pledges grant bronze, silver, or gold keys which can be used to open treasure chests within the enclaves which contain valuable set pieces.

Monster Helm Sets

The first special type of set which can be acquired through dungeons are "monster helms", which are sets that are thematically based on iconic bosses from each dungeon. These monster helms represent both powerful equipment and trophies, and the helms from each set are acquired by slaying the namesake boss within each set's respective dungeon. The helm itself drops directly from the boss it represents. The shoulder piece from each set comes as a potential reward from the Undaunted pledge chests, so it's important to do daily pledges in order to give yourself a chance of acquiring the piece you need.

The item set bonus values in orange are variable depending on the item level and item quality. Some set bonuses like weapon or spell critical are static, however. All Veteran Dungeons prior to City of Ash cap at a maximum veteran rank of VR12, meaning that the helm piece which is dropped by the boss within the dungeon has a maximum possible level of VR12. The shoulders which come from the pledge chests are scaled to your character's Veteran Rank, and thus can be obtained up through VR14. The set bonus values listed are scaled for VR14, but most completed sets will be comprised of a VR12 head piece and a VR14 shoulder piece. In this case, the player receives about 131 Max Health instead of just 129 (VR12) or 135 (VR14).

It is important to note that all Monster Helm sets are random in both their armor type (Light/Medium/Heavy) as well as their trait (Reinforced, Infused, Divines, etc...). In order to get the "perfect" set, players will need to put a lot of time into farming and hope for a bit of luck on their pledge chest rolls!

Undaunted Sets

A second type of armor set available since Update 5 are the Undaunted armor sets, which are three complete sets of Light, Medium, and Heavy armor which thematically represent the Undaunted guild style. These sets are designed with 5 piece bonuses which encourage hybridization and create opportunities for more interesting combinations of ability usage.

These armor sets are only be obtained by completing the daily pledges and opening the chests. Gold Chests have a chance on purple Undaunted gear and Silver Chest have a chance on blue Undaunted gear. Much like the Monster Helm sets, the traits which come on Undaunted armor pieces are randomized, so it's possible to get any of the combat traits for each set piece. Unlike the Monster Helms, the armor type of each set is fixed, for example Undaunted Unweaver is only ever found as Light Armor.

City of Ash Sets

The third type of armor set added in Update 5 are three armor sets unique to the new Veteran mdoe of the City of Ash dungeon. Like Undaunted armor sets, these always come in a pre-defined type. For example the "Burning Spellweaver" set is always found as Light Armor. It is (as of yet) unknown whether the traits on these pieces are randomized or static. We'll update this page in the near future with a definitive answer regarding which traits are possible for these three sets.
